# 스크린샷 OCR & DB 기록 계획서

## 개요
- **목적**
1. 킹샷 게임에 존재하는 연맹원 데이터 관리 및 이를 정제해서 보여줄 수 있는 페이지 생성.
2. 스크린샷에서 텍스트를 추출하여 특정 이벤트에 참가한 유저들 식별, 점수 기록, 데이터 트래킹 기능 필요
3. 킹샷에 대해 웹 검색을 통해 기본적인 정보들을 취합해서 유저 구조 수립
    - 킹샷은 각 유저별로 8자리 integer 형태의 고유 id 존재
    - 킹샷 API 중 위 id 를 바탕으로 조회시 nickname 과 각종 메타 데이터를 불러올수있는것이 있음. 서치 해서 응답값을 확인하고 해당 데이터들을 모두 담아둘수 있게끔 유저 모델 수립
    - 회원가입은 저 id 와 설정할 비밀번호, 비밀번호 확인 을 받아서 API 호출후 모델 각 필드값들을 채우는 형태로 진행
